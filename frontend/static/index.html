                    Shortcuts
                    
Keyboard flow

                  

                  
Hold Ctrl anywhere in the launcher to reveal these hints directly on the UI.

                  

                    
TabMove focus through installed versions and controls.

                    
Shift+TabMove focus backward.

                    
EnterActivate the focused version or button.

                    
Ctrl+FFocus version search.

                    
F8Open Add Version.

                    
F9Launch the selected version.

                    
F2Open or close Settings.

                    
Ctrl+SSave Settings while the settings page is open.

                    
EscClose settings or dialogs.
